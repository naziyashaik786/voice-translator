# ProjectPilot AI

Welcome to the ProjectPilot AI codebase. This repository contains both the React frontend and the Express backend.

## Structure
- `/frontend`: React + Vite + Tailwind CSS app.
- `/backend`: Node.js + Express + TypeScript API.

## How to Run

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
1. Copy `backend/.env.example` to `backend/.env` and fill in the values.
```bash
cd backend
npm install
npm run dev
```
